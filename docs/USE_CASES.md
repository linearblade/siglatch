# Siglatch Use Cases

Siglatch fits infrastructure where you want authenticated UDP control with a small footprint and tight policy boundaries.

## SSH Access Management
Grant or revoke specific IPs the ability to reach protected systems.

## Web Service Control
Toggle dashboards, admin panels, or internal UIs on and off as needed.

## Dead-Drop Communication
Exchange raw or structured payloads with minimal metadata.

## Interserver Signaling and Command Dispatch
Push secure instructions like `reload`, `rotate`, `notify`, or other daemon actions.

## Log Forwarding and Job Scheduling
Replace syslog chains and cron glue with encrypted packet-based triggers.

## Custom Action Handlers
Route requests into shell scripts, static objects, or dynamic objects for specialized behavior.

## Protocol and Transport Experiments
Use user-defined UDP protocol families and demux multiple wire formats on the same daemon.
